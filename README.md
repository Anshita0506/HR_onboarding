# HR_onboarding


This project automates the employee onboarding process using **Microsoft Excel** and **Power Automate**. It reduces manual HR effort by streamlining routine tasks such as sending welcome emails, notifying managers, and updating onboarding status.

###  Key Highlights
- Automated onboarding triggered from Excel entries
- Personalized welcome and notification emails
- Real-time status updates within the Excel sheet
- Works with personal OneDrive using a scheduled flow

### Tools Used
- Microsoft Excel
- Power Automate
- Outlook (for email actions)

###  Included
- Sample Excel file with 200 dummy employee records
- Recurrence-based automation design compatible with personal Microsoft accounts

---

> This project improves HR communication, saves time, and brings consistency to the onboarding workflow.
